# NosAlive_Clone
This web project is to build a clone of 'Nos Alive' website using HTML, CSS, Javascript, Java/JSP and MySQL in 2016.

## Objectives and Functions

Website that provides information for Nos Alive music festival 

C/R/U/D for members
